Renewable Energy

GitHub

This site was created to demo several D3 charts and graphs developed as part of the "Data Visualization and Inforgaphics with D3" online course provided by JournalismCourses.org.

The data set consisted of 41 countries and the percentage of their Renewable Energy production 2002 to 2012.


Program Overview Dashboard

GitHub

This dashboard utilizes the following D3 charts\graphs and functionality

  • World Map plotting the cities participating in the program
  • Stacked Bar Chart displaying the regional program status
  • Pie Chart displaying total's of program status categories
  • Interactive Legend that filters one or more choices

The site is currently hosted on heroku and was built using Node, Express and Bootstrap.
